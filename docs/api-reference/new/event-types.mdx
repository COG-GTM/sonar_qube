# 이벤트 타입 API

이벤트 타입은 예약 가능한 시간 슬롯의 템플릿입니다. 이벤트 타입을 통해 미팅 시간, 기간, 설명 등을 정의할 수 있습니다.

## 이벤트 타입 목록 조회

### HTTP 요청
```http
GET /v1/event-types
```

### 쿼리 파라미터
| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| apiKey | string | 예 | API 키 |

### 응답 예시
```json
{
  "event_types": [
    {
      "id": 1,
      "title": "30분 미팅",
      "slug": "30min",
      "length": 30,
      "description": "짧은 미팅",
      "hidden": false,
      "position": 0
    }
  ]
}
```

## 이벤트 타입 생성

### HTTP 요청
```http
POST /v1/event-types
```

### 요청 본문
```json
{
  "title": "60분 미팅",
  "slug": "60min",
  "length": 60,
  "description": "긴 미팅",
  "hidden": false,
  "teamId": null,
  "schedulingType": "ROUND_ROBIN"
}
```

### 필수 필드
- title: 이벤트 타입 제목
- length: 미팅 시간(분)

## 이벤트 타입 수정

### HTTP 요청
```http
PATCH /v1/event-types/{id}
```

### 경로 파라미터
| 파라미터 | 타입 | 설명 |
|----------|------|------|
| id | integer | 이벤트 타입 ID |

### 요청 본문
수정하고자 하는 필드만 포함하면 됩니다.

## 이벤트 타입 삭제

### HTTP 요청
```http
DELETE /v1/event-types/{id}
```

### 경로 파라미터
| 파라미터 | 타입 | 설명 |
|----------|------|------|
| id | integer | 이벤트 타입 ID |

## 오류 코드
| 코드 | 설명 |
|------|------|
| 400 | 잘못된 요청 형식 |
| 401 | 인증 정보 누락 또는 잘못됨 |
| 404 | 이벤트 타입을 찾을 수 없음 |
